# Hyperbolicity-Dimension-2-2026-Code
A collection of codes and files for new vanishing results for invariant (logarithmic) $2$-jet differentials in dimension $2$

##  Associated Paper

This repository contains the computational materials and Maple codes supporting the results presented in the following paper:

> **arXiv:2603.14881**
> **Title:** IMPROVED DEGREE BOUNDS FOR HYPERBOLICITY OF SURFACES AND CURVE COMPLEMENTS

##  Repository Contents

This repository primarily includes:

-   **compact:** Contains the subdirectories `fact-cpt`, `key-vanishing-cpt`, and `nonvanishing-example-cpt` related to the compact case calculations.
    -   `fact-cpt`: Contains PDF files which are exports generated by the code `key_vanishing_compact_maple.mw` (located in `key-vanishing-cpt`) with adjusted parameters $d, m, t$. These correspond to the **Fact** in **Remark 6.3** of the associated paper.
    -   `key-vanishing-cpt`: Includes the Maple code `key_vanishing_compact_maple.mw` used for the proof of **Lemma 1.4 (Key Vanishing Lemma– Compact Case)** (see **Subsection 4.6**), along with the explanation file `key_vanishing_compact_explanation.pdf` and the export file `key_vanishing_compact_results.pdf`.
    -   `nonvanishing-example-cpt`: Contains exports of nonvanishing results obtained by adjusting the surface equation and parameters $d, m, t$ in the `key_vanishing_compact_maple.mw` code.
-   **logarithmic:** Contains the subdirectories `fact-log`, `key-vanishing-log`, and `nonvanishing-example-log` related to the logarithmic case calculations.
    -   `fact-log`: Contains PDF files which are exports generated by the code `key_vanishing_log_maple.mw` (located in `key-vanishing-log`) with adjusted parameters $d, m, t$. These correspond to the **Fact** in **Remark 6.4** of the associated paper.
    -   `key-vanishing-log`: Includes the Maple code `key_vanishing_log_maple.mw` used for the proof of **Lemma 1.3 (Key Vanishing Lemma– Logarithmic Case)** (see **Subsection 3.3**), along with the explanation file `key_vanishing_log_explanation.pdf` and the export file `key_vanishing_log_results.pdf`.
    -   `nonvanishing-example-log`: Contains exports of nonvanishing results obtained by adjusting the curve equation and parameters $d, m, t$ in the `key_vanishing_log_maple.mw` code.
-   **conventions:** Contains the subdirectories `compact` and `logarithmic` verifying conventions used in the computations.
    -   `compact`: Includes the Maple code `convention cpt.mw` used to verify **Convention 4.16** in the associated paper, along with the export file `convention cpt.pdf`.
    -   `logarithmic`: Includes the Maple code `convention log.mw` used to verify **Convention 3.3** in the associated paper, along with the export file `convention log.pdf`.

These files are intended to reproduce and verify the calculations found in the associated mathematical paper.

##  Usage

1.  Download the specific `.mw` files directly (using the "Download raw file" button on GitHub) or clone the repository to your local machine.
2.  Open the `.mw` files within the Maple environment.
3.  Execute the worksheets to regenerate the exports or modify parameters for further experimentation.
